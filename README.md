# aishouji-ssm_e-store，网上二手机售卖

### 一、使用技术
###### Java、JQuery、BootStrap、jsp、Vue基础、Spring、SpringMVC、MyBatis。

### 二、借鉴
###### 一些页面CSS借鉴了找靓机、京东网页布局。

### 三、开发环境
###### idea，使用Chrome、火狐浏览器，使用tomcat9，MySQL5.x、JDK1.8。

### 四、功能
###### 登录，注册、找回密码、加入购物车、提交购买、查看购物车、查看订单。

###### 数据库文件已提供，aishouji_db.sql。
